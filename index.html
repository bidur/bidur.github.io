Education and Training
Doctor of Engineering, Department of Information and Communication Technologies (ICT) August 2016 – July 2020
Asian Institute of Technology, Thailand

Research: A Study on Tourism Monitoring using Social Media and Nighttime Light Remote Sensing Data

This work focuses on extracting crowdsourced data from online sources like Twitter, Flickr and OpenStreetMap as well as remote sensing data and analyze them to identify and describe tourism areas (study area: Nepal). Data analysis is done using state-of-the-art spatial clustering and text mining algorithms like DBSCAN, TFIDF, etc. Open-source tools like Python and QGIS are used for extracting and analyzing the data.

Asian Institute of Technology, Thailand

Thesis: Directed Accident Information Dissemination in Vehicular Ad-hoc Network

This work proposes an approach for disseminating alert messages among vehicles on the road, for city and highway conditions, in a directional manner so as to ensure safe and improved road experience.

Research Area: Machine Learning, Text Mining, Social Media Mining

This work focuses on extracting hotel review data from travel portals like TripAdvisor to identify important hotel aspects and their weight so as to rank the hotels based on the discovered aspects and their rank. State-of-the-art algorithms like Latent Dirichlet Allocation and Word2Vec were used for the purpose.

Python programming language is used for extracting and analysing the data.
